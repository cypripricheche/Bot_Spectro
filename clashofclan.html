Config Clash Of Clans ╶╴ Partie 1

Pour commencer, vous allez devoir exécuter la commande /config_clashofclan et entrer le # du clan que vous souhaitez paramétrer.
Spectro vous affichera ensuite l'embed avec toutes les paramètres en lien avec Clash Of Clan.

Aperçu de la commande /config_clashofclan

Config Clash Of Clans ╶╴ Partie 2

Après avoir exécuté la commande /config_clashofclan, vous recevrez un embed vous permettant de configurer les salons de votre clan.
Pour les guerres, cliquez simplement sur le bouton Config Guerre et saisissez le # du salon concerné.

Résultat de la commande /config_clashofclan

Config Clash Of Clans ╶╴ Guerre

Après avoir cliqué sur le bouton Config Guerre, vous recevrez un embed concernant votre guerre dans les deux prochaines minutes.
Vous disposez également d'un fil contenant le récapitulatif des scores de guerre, vous permettant de suivre en temps réel les performances des membres de votre clan.

Résultat de la configuration des guerres.

Config Clash Of Clans ╶╴ Rappel Guerre

En cliquant sur le bouton Config Rappel, vous pourrez configurer les rappels de guerre et recevrez un message indiquant les attaques restantes.
⚠️ Cette fonctionnalité est disponible uniquement avec l'option Premium.

Résultat de la configuration des rappels de guerres.

Compte Clash Of Clans ╶╴ Partie 1

Pour recevoir les rappels de guerre, vous allez devoir exécuter la commande /compte_clashofclan, afin d'enregistrer vos comptes.
⚠️ Cette fonctionnalité est disponible uniquement avec l'option Premium.

Résultat de la configuration des rappels de guerres.

Compte Clash Of Clans ╶╴ Partie 2

Après avoir exécuté la commande /compte_clashofclan, cliquez sur Enregistrer Compte pour le relier à Spectro.
⚠️ Cette fonctionnalité est disponible uniquement avec l'option Premium.

Résultat de la configuration des rappels de guerres.
